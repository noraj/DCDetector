DC Detector

Spot all domain controllers in a Microsoft Active Directory environment.

CLI

dcd -d za.tryhackme.com -s 10.200.28.101

Library

Features

  • CLI and library
  • Find computer name, FQDN, and IP address(es) of all DCs
